# RustEgg Releases

Релизы кастомного клиента Minecraft и лаунчера RustEgg

## 🎮 Компоненты

- **Client**: Minecraft 1.12.2 с LWJGL 3.4.1 и OpenGL 4.1+
- **Launcher**: Автоматическая установка и обновления

## 🚀 Особенности клиента

- ✅ LWJGL 3.4.1 для современной графики
- ✅ OpenGL 4.1+ поддержка
- ✅ Интеграция карт в стиле Rust
- ✅ Оптимизация производительности
- ✅ Поддержка Java 17+

## 📥 Скачивание

1. Скачай [RustEggLauncher.exe](https://github.com/artikhupix/rustegg-releases/releases/latest)
2. Запусти лаунчер
3. Введи никнейм и настрой параметры
4. Нажми "ИГРАТЬ" - клиент скачается автоматически!

## ⚙️ Системные требования

- **Java 17+** (лаунчер поможет установить)
- **OpenGL 4.1+** поддержка
- **4GB RAM** минимум
- **Windows/Linux/macOS**
